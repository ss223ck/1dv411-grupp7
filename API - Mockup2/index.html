Detta är en test sida för att simulera en request via vår sida till en API.

Resultatet som vi får från API:et visas i konsolen, om man använder sig av hämta knappen under.

Vi frågar efter användarnas GitHub användarnamn via anrop med t.ex lnu användarnamn.
